    Keyboard Shortcuts¶

    Use kbd tags for keyboard shortcuts:

    • Press :kbd:Ctrl + :kbd:C to copy
    • Use :kbd:Ctrl + :kbd:V to paste
    • :kbd:Ctrl + :kbd:S saves the file
    • :kbd:Ctrl + :kbd:F opens search
    • :kbd:Ctrl + :kbd:Z undoes the last action
